Vartalabh — Update Timeline

Release & Update Report

Vartalabh is built for fast and meaningful conversations. Here’s what improved across major versions and how the platform evolved from core messaging to a faster, modern experience.

Latest Version — v3.1.0

Modern messaging improvements
  • Group Chat for smoother multi-user conversations.
  • Voice Messages support for quick audio communication.
  • Image Sharing in Chat for richer interactions.
  • Push Notifications for real-time message awareness.
  • Seen / Delivered Status for message tracking.
  • Unread Badge Counts for quick inbox visibility.
  • Username Search to discover users faster.
  • Block User option to prevent unwanted chats.
  • Report User system for safer community behavior.
  • Delete for Me / Delete for Everyone controls.
  • Faster chat loading and improved UI.

Previous Version — v2.3.0

Earlier messaging upgrades
  • Improved one-to-one chat performance and reliability.
  • Better message delivery feedback in active conversations.
  • Enhanced unread message visibility in chat lists.
  • Cleaner conversation layout and improved readability.
  • Profile and username handling refinements.
  • Notification behavior tuned for more stable alerts.

Foundation Version — v1.2.0

Core features that started it all
  • Core user-to-user messaging support.
  • Basic account and identity setup for chat access.
  • Initial conversation list and message thread screens.
  • Early send/receive message pipeline.
  • First implementation of lightweight mobile chat UX.
